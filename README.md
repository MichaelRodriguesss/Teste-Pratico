## :rocket: Tecnologias

## :checkered_flag: Iniciando o Projeto

```bash
# Clone this repository
$ git clone https://github.com/MichaelRodriguesss/Teste-Pratico.git

# Install dependencies
$ npm install

# Run the app
$ npm run dev
```
